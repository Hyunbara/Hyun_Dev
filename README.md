# 개발자 블로그 프로젝트 🚀

## 소개
나를 소개하는 개발자 블로그를 만드는 프로젝트입니다.

## 기술 스택

### Frontend
- Next.js (v15.3.2)
- React (v19.0.0) 
- TailwindCSS (v4)
- TypeScript (v5)

### 개발 도구
- Biome (v1.9.4) - 린팅 & 포맷팅

### Backend & 배포
- Firebase API - 데이터 CRUD 작업
- Vercel - 배포 플랫폼

### 추가 라이브러리
- **폼 관리 & 유효성 검사**
  - Zod
  - React Hook Form
- **UI/UX**
  - next-themes (다크모드)
  - shadcn/ui (UI 컴포넌트)
- **SEO & 콘텐츠**
  - next/head (SEO, OG 태그)
  - Markdown 지원
